## Автотесты для конструктора календаря мероприятий 3Snet

Проект с автотестами на **JavaScript + Playwright** для страницы-конструктора календаря мероприятий:

- URL: `https://dev.3snet.info/eventswidget/`

Тесты используют `@playwright/test`, конфигурация находится в `playwright.config.js`, логика страницы вынесена в Page Object `src/pages/mainPage.js`, сами тесты — в `tests/eventswidget.spec.js`.

### ПО для работы с проектом

Перед началом работы убедитесь, что на вашем устройстве установлены следующие ПО:

- **Инструмент разработки**: IntelliJ IDEA, VS Code или другой IDE/редактор.
- **Git**.
- **Браузер**: Google Chrome (для удобства анализа и отладки).
- **Node.js** (рекомендуется версия не ниже 18) и менеджер пакетов **npm**.

### Установка и запуск

1. **Склонировать репозиторий на свой рабочий стол (или в любую другую директорию)**:

```bash
git clone git@github.com:DmitriyRonMan/test_for_3snet.git
cd test_for_3snet
```

2. **Установить зависимости проекта**:

```bash
npm install
```

3. **Установить браузеры Playwright** (однократно, если ещё не установлены):

```bash
npx playwright install
```

4. **Запустить тесты**:

```bash
npx playwright test
```

5. **Получить HTML-отчёт о прогоне тестов**:

```bash
npx playwright show-report
```

После выполнения этих шагов проект полностью готов к локальному запуску и анализу результатов автотестов.
